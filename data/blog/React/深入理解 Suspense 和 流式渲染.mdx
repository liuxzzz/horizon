## Suspense

### Suspense 的官方定义
`<Suspense>` 允许在子组件完成加载前展示后备方案。

```
<Suspense fallback={<Loading />}>
  <SomeComponent />
</Suspense>
```

后备方案，就是`fallback` 是 `Suspense` 的属性，简单来说就是在子组件加载完成前显示的loading 或者是占位符。
